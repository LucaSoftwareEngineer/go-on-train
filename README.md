# GO ON TRAIN
## Descrizione
### Applicativo che permette agli utenti di registrarsi e accedere ad un portale in cui hanno la possibilità di visualizzare l'elenco dei treni e di prenotarne uno. L'applicativo permette agli amministratori pre-registrati nel sistema di registrare le corse dei treni, quindi data e ora di partenza e di arrivo a destinazione.
